# my-informaciyniy-site
Детальне пояснення до проєкту з демонтсрацією сайту https://docs.google.com/document/d/1uCK7IJewlZm1JMrcB1DFS_CI5tUKx-Av/edit?usp=sharing&ouid=102215451454217917429&rtpof=true&sd=true
Архів сайту на Google Диску: https://drive.google.com/file/d/115or17g22jMuhLjUbptvD5m9wGfN_4Hm/view?usp=drive_link
У зв'язку з обмеженням GitHub на розмір файлів при завантаженні через браузер (понад 25 МБ), проєкт розділено на дві частини:
1. База даних проєкту: Файл site.sql завантажено безпосередньо сюди в репозиторій.
2. Файловий архів сайту (папка з htdocs): Повний ZIP-архів сайту (близько 50 МБ) розміщено на Google Диску за посиланням вище.
   
У репозиторій додано архітектурні файли шаблону, вихідний код навігації та експорт візуального дизайну:
1. Логіка шаблону та меню
* шаблон/ — папка, що містить основні архітектурні файли структури сайту:
  * index.php — головний файл структури та модульних позицій шаблону.
  * component.php — файл для виводу окремих компонентів (вікна друку, спливаючі вікна).
  * error.php — кастомна сторінка обробки помилок сайту (404).
* menu.php — вихідний PHP-код, який відповідає за генерацію та рендеринг навігаційного меню сайту.
* default.php — кастомний PHP-код відображення мовного модуля (активація та налаштування системного мовного плагіна в БД).
2. Візуальний дизайн та верстка сторінок (SP Page Builder)
Додано файли структури головної сторінки у форматі JSON (повний бекап візуальної верстки):
* sp-page-builder-export UK.json — вихідний код архітектури та дизайну головної сторінки проєкту українською мовою.
* sp-page-builder-export EN.json — вихідний код архітектури та дизайну головної сторінки проєкту англійською мовою.
